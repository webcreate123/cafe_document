# 画面遷移図（テキスト版）

## 1. `cafe_web`（利用者/店舗）

### 1.1 共通導線
```text
トップ (/)
 ├─ ログイン (/auth/login)
 │   ├─ 会員登録 (/auth/register)
 │   │   └─ 登録完了 (/auth/register-success)
 │   ├─ パスワード再発行申請 (/auth/password-reset)
 │   │   └─ パスワード再設定 (/auth/reset-password)
 │   └─ OAuthコールバック (/auth/oauth-callback)
 ├─ マップ (/map)
 │   └─ 店舗詳細 (/stores/[id])
 └─ マイページ (/mypage)
     └─ 設定 (/settings)
```

### 1.2 一般ユーザー導線
```text
マイページ (/mypage)
 ├─ お気に入り一覧 (/user/favorites)
 │   └─ お気に入り詳細 (/user/favorites/[id])
 ├─ 閲覧履歴 (/user/history)
 ├─ 求人一覧 (/user/jobs)
 │   ├─ 求人詳細 (/user/jobs/[id])
 │   └─ お気に入り求人 (/user/jobs/favorites)
 └─ 決済情報 (/user/payments)
     └─ 支払い方法 (/user/payments/methods)
```

### 1.3 店舗ユーザー導線
```text
店舗ダッシュボード (/store/dashboard)
 ├─ 店舗設定 (/store/settings)
 │   ├─ 管理店舗一覧 (/store/settings/stores)
 │   └─ 権限設定 (/store/settings/permissions)
 ├─ お知らせ一覧 (/store/announcements)
 │   ├─ 新規投稿 (/store/announcements/new)
 │   └─ 編集 (/store/announcements/[id])
 ├─ セール一覧 (/store/sales)
 │   ├─ 新規投稿 (/store/sales/new)
 │   ├─ 編集 (/store/sales/[id])
 │   └─ テンプレート管理 (/store/sales/templates)
 ├─ 求人一覧 (/store/jobs)
 │   ├─ 新規作成 (/store/jobs/new)
 │   └─ 編集 (/store/jobs/[id])
 ├─ 問い合わせ (/store/inquiry)
 └─ 領収書 (/store/payments/receipts)
```

## 2. `cafe_admin`（管理）

```text
ログイン (/login)
 └─ ダッシュボード (/dashboard)
     ├─ ユーザー管理 (/users)
     ├─ 店舗管理 (/stores)
     ├─ 店舗承認 (/stores/approval)
     ├─ 通報管理 (/reports)
     ├─ 決済管理 (/payments)
     └─ 分析 (/analytics)
```

## 3. 補足
- `cafe_web`は`[locale]`付きルートで提供（例: `/ja/map`, `/en/map`）。
- 店舗登録申請は`/stores/register`から開始し、承認後に店舗管理導線へ遷移可能。
