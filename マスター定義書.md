# マスター定義書（Cafe Platform）

## 1. 目的
本書は、アプリケーション内で固定値として扱うコード体系（ロール、状態、分類、イベント種別等）を定義する。

## 2. ユーザー関連

### 2.1 ユーザーロール（`UserRole`）
| 値 | 意味 |
| --- | --- |
| `USER` | 一般利用者 |
| `STORE_OWNER` | 店舗オーナー |
| `STORE_STAFF` | 店舗スタッフ |
| `ADMIN` | 管理者 |

### 2.2 ユーザー状態（`UserStatus`）
| 値 | 意味 |
| --- | --- |
| `DEFAULT` | 既定 |
| `PENDING` | 検討中 |
| `STOPPED` | 停止 |
| `REJECTED` | 拒否 |

## 6. 通知・イベント関連

### 6.1 通知種別（`NotificationType`）
| 値 | 意味 |
| --- | --- |
| `new_store_nearby` | 近隣新着店舗 |
| `sale_announcement` | セール告知 |
| `job_application` | 求人応募関連 |
| `system` | システム通知 |

### 6.2 イベント種別（`EventType`）
| 区分 | 値 |
| --- | --- |
| ユーザー | `user_registered`, `user_login`, `user_logout`, `user_profile_updated` |
